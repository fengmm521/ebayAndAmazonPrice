使用到的第三方python库

bs4,pil,dbm

程序使用python版本,大于2.7.10,小于3.0.0

服务器配置文件config.txt说明:
文件为json格式，
{
    "ip":"你的服务器内网ip",
    "port":服务器监听端口,
    "user":{
        "用户名1":"用户1的密码",
        "用户名2":"用户2的密码"
    }
}
ip为你的服务器的内容监听ip
port为服务器监听端口
user为用户名和用户密码
    用户名:密码

文件说明,
BS4HTMLTool.py主要功能为:
1.从ebay和amazon获取html文档
2.使用bs4库分析html得到图片地址，商品名称和商品价格(美元价格)
3.可存得到了数据到db/data.db数据库中
4.下载商品图片并缩小为80x80，保存在img目录下，以供浏览器请求时显示

dbTool.py主要功能:
一个轻量级的key-value数据库，用来保存数据

httpsServer.py主要功能:
1.http服务器功能，响应浏览器的http常规请求和表单提交等所有http请求，并提供图片http服务器功能
2.启动BS4HTMLTool.py下载数据程序
3.每两小时调用一次BS4HTMLTool.py的更新数据方法来更新数据
4.从db数据库中的数据生成相关商品列表html文档返回给浏览器

html目录中的html文件：
主要为要显示的所有页面

db目录：
保存所有下载的价格，更新时间，图片下载地址，商品访问地址，价格数据

img目录：
保存下载到的商品图片
